uAdd - утилита добавления .torrent файлов к uTorrent WebUI
Автор: uaddproject@gmail.com
Проект: http://github.com/uaddproject/uadd

Как настроить.

Запустить программу uTorrent (протестировано на версии 3.3.2). Перейти в меню "Настройки" >> "Настройки программы".Далее в ветку настроек "Дополнительно" >> "Веб-интерфейс".
Установить флажок "Использовать веб-интерфейс", указать имя и пароль. При необходимости можно разрешить гостевой вход с указанным именем или указать альтернативный порт. 
Также при необходимости можно указать список доступных ip-адресов клиентов.
Далее самое важное для программы uAdd. Заполните раздел "Папки загрузки". 
Здесь указываются каталоги на диске сервера (компьютер на котором запущена программа uTorrent), которые будут являться категориями для утилиты uAdd.

Пример настроек.
Программа uTorrent (сервер):

Флажок "Использовать веб-интерфейс": Да
Имя учетной записи: admin
Пароль учетной записи: adminpass
Флажок "Использовать гостевой вход с именем": guest
Флажок "Альтернативный порт": Да
"Альтернативный порт": 8081

Группа "Папки загрузки":
"D:\Share\Кино"
"D:\Share\Игры"
"D:\Share\Сериалы"
"D:\Share\Мультфильмы"
"D:\Share\Книги"

Программа uAdd (клиент):
Запустить программу uAdd. Перейти в "Настройки".
В блоке "Настройки uTorrent gui сервера" заполнить. 
Сервер:  127.0.0.1:8081
Где "127.0.0.1" - ip адрес сервера на котором запущена программа uTorrent, а "8081" - порт указанный в настройках uTorrent (сервер).
Пользователь: admin
Пароль: adminpass
Далее нужно проверить соединение с сервером нажав кнопку "Тест".
Далее перейти к блоку настроек "Категории". В данном разделе заполняется соответствие категории (группа, метка ) контента и каталога на диске сервера, куда контент будет скачен.
Таблицу категория можно заполнить 3 способами.
1. Автоматически загрузить с сервера нажав кнопку "Загрузить из uTorrent gui сервера" на панели управления таблицы. Это возможно только при подключенном сервере.
2. Загрузить данные из файла данных настроек программы uTorrent. Файл называется "settings.dat"  находится в той же папке что и программа uTorrent на сервере.
   Для данной загрузки необходимо нажать кнопку "Загрузить из Settings.dat" на панели управления таблицы.
3. Заполнить таблицу вручную. 
   Колонка "Категория" - это имя категории которая будет отображаться в программе uAdd.
   Колонка "№ каталога" - это порядковый номер каталога в настройках программы uTorrent на сервере.
   Колонка "Комментарий" - комментарий для удобства заполнения.

На основе данных примере при автоматическом заполнении получится следующее:

Категория   | № каталога |  Комментарий
Кино        | 1          |  D:\Share\Кино
Игры        | 2          |  D:\Share\Игры
Сериалы     | 3          |  D:\Share\Сериалы
Мультфильмы | 4          |  D:\Share\Мультфильмы
Книги       | 5          |  D:\Share\Книги
